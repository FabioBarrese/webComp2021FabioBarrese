Cosa imparerai su questo sito?

Questo sito è nato con lo scopo di trasmettere al lettore le competenze di base per sviluppare videogiochi 2D e per rendere il proprio sito web interattivo; saranno quindi presenti diversi articoli riguardanti i linguaggi di programmazione Java e JavaScript. Lo sviluppo di videogiochi si baserà sul framework libGDX mentre la programmazione web lato client si baserà sull'utilizzo del framework p5.js.
